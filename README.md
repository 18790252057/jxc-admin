## 简介

一个前后端分离的简易进销存后台管理系统，基于 [SpringBoot](https://spring.io/projects/spring-boot/) 和 [vue-element-admin](https://github.com/PanJiaChen/vue-element-admin)实现，具备常见的后台管理功能，后端另有SpingCloud版本。
